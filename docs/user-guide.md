## 1. 시작하기

### 1-1. 회원가입 및 로그인

1. 홈페이지에서 **[로그인]** 버튼 클릭
2. 다음 중 하나를 선택하여 가입 진행:
   - **Google로 로그인** — 구글 계정 사용
   - **Kakao로 로그인** — 카카오 계정 사용
3. 최초 가입 시 이용약관/개인정보처리방침 동의 필요

![로그인 화면](./screenshots/login.png)

> 주의: 만 14세 미만 아동은 가입할 수 없습니다.

### 1-2. 계정 설정

우측 상단 프로필 아이콘 → **[계정 설정]** 에서:

- **이름** 수정
- **구독 플랜** 확인 및 해지
- **로그아웃**

## 2. 대시보드

로그인 후 기본 진입 페이지입니다.

- **최근 연구 노트** — 최근 열어본 노트 바로가기
- **이번 달 사용량** — 플랜 한도 대비 연구 노트 생성 수
- **[+ 새 연구 시작]** — 새 노트로 이동

![대시보드](./screenshots/dashboard.png)

---

## 3. 논문 검색

### 3-1. 기본 검색

좌측 사이드바 **[논문 검색]** 또는 연구 노트 상단 검색창에서:

1. 검색어 입력(키워드, 제목, 저자)
2. **Enter** 또는 검색 버튼
3. arXiv · Semantic Scholar 등 여러 소스에서 통합 결과 제공

### 3-2. 필터

검색 결과 우측 **[필터]** 버튼 → 다음 조건 적용:

| 필터 | 설명 |
| --- | --- |
| 연도 범위 | 시작년도 ~ 종료년도 |
| 소스 | arXiv, Semantic Scholar, DBLP 등 |
| 인용 수 | 최소 인용 수 |

### 3-3. 북마크

논문 카드의 ⭐ 아이콘을 클릭하면 개인 북마크에 저장됩니다. 북마크는 **[북마크]** 메뉴에서 일괄 관리할 수 있습니다.

![논문 검색](./screenshots/search.png)

## 4. 컬렉션 관리

주제별로 논문을 묶어 관리하는 기능입니다.

### 4-1. 컬렉션 생성

1. **[컬렉션]** 메뉴 → **[+ 새 컬렉션]**
2. 이름 · 설명 입력 → **[저장]**

### 4-2. 논문 추가

1. 논문 검색 결과 카드에서 **[컬렉션에 추가]** 클릭
2. 기존 컬렉션 선택 또는 신규 생성
3. 태그 입력(선택) — 예: `#LLM`, `#서베이`

### 4-3. 컬렉션에서 제거

컬렉션 상세 페이지 → 논문 카드 우측 **×** 버튼

![컬렉션](./screenshots/collections.png)

## 5. 연구 노트 작성 & 자동저장

### 5-1. 새 노트 시작

대시보드 **[+ 새 연구 시작]** 또는 **[연구]** 메뉴.

### 5-2. 자동저장

- 편집 중지 **3초 후** 자동으로 서버에 저장됩니다(debounce).
- 저장 상태는 상단 표시줄에서 확인 가능:
  - "저장 중…" → 편집 진행
  - "저장됨 · 방금 전" → 저장 완료
- 수동 저장은 **Ctrl + S** (macOS: **Cmd + S**)

### 5-3. 참고문헌 삽입

1. 에디터 내 `@` 입력 → 북마크/컬렉션에서 논문 검색
2. 선택 시 본문에 인용 마크 삽입, 하단 참고문헌 목록 자동 갱신

![연구 노트](./screenshots/research.png)

## 6. 버전 관리

연구 노트는 저장 시점마다 자동으로 버전이 생성됩니다.

- **버전 보기**: 노트 우측 상단 🕘 아이콘
- **버전 비교**: 두 버전을 선택하면 diff 뷰 표시
- **이전 버전으로 복원**: 버전 상세에서 **[이 버전으로 복원]**

> 수동 저장(`Ctrl+S`)은 명시적 버전(`save_type=manual`), 자동저장은 `save_type=auto` 로 기록됩니다.

## 7. Export (BibTeX, Markdown, PDF)

노트 우측 상단 **[내보내기]** 드롭다운:

| 형식 | 용도 |
| --- | --- |
| **Markdown** | GitHub/Obsidian 등 |
| **BibTeX** | LaTeX 참고문헌 |
| **PDF** | 인쇄 · 제출용 |
| **DOCX** | MS Word |

> Export는 비동기로 처리됩니다. 완료되면 브라우저 알림과 함께 다운로드 링크가 표시됩니다.

![Export](./screenshots/export.png)

## 8. 설문 생성

AI를 사용해 연구 주제에 맞는 설문 문항을 자동 생성합니다.

1. **[설문]** 메뉴 → **[+ 새 설문]**
2. 연구 주제 · 대상자 · 문항 수 입력
3. **[생성]** 클릭 → 수 초 내 초안 생성
4. 문항별 수정 · 삭제 · 순서 변경 가능
5. **[공유 링크 생성]** 으로 응답자에게 배포

## 9. 플랜 & 결제

| 플랜 | 가격(월) | 월 연구 노트 | AI 요약 | Export |
| --- | --- | --- | --- | --- |
| Free | 0원 | 3개 | 제한 | Markdown만 |
| **Basic** | 9,900원 | 30개 | 무제한 | 전 형식 |
| **Pro** | 29,900원 | 무제한 | 무제한 | 전 형식 + 우선 처리 |

### 9-1. 결제 방법

1. **[플랜 · 결제]** 메뉴 → 원하는 플랜 **[업그레이드]**
2. PortOne(아임포트) 결제창에서 카드/간편결제 선택
3. 결제 완료 후 즉시 플랜 반영

### 9-2. 구독 해지

**[계정 설정]** → **[구독 취소]**. 해지 후에도 다음 결제일까지는 유료 기능을 이용할 수 있습니다.

### 9-3. 환불

- 결제 후 **7일 이내 · 유료 기능 미사용** 시 전액 환불
- 자세한 내용은 [환불정책](../frontend/app/refund/page.tsx) 참조

## 10. 자주 묻는 질문

**Q1. 동시에 여러 기기에서 로그인할 수 있나요?**
A. 가능합니다. 세션은 기기별로 관리됩니다.

**Q2. 데이터 백업은 어떻게 하나요?**
A. 연구 노트는 언제든 Markdown/PDF로 Export하여 로컬에 보관하실 수 있습니다.

**Q3. AI가 생성한 문장의 저작권은 누구에게 있나요?**
A. 이용자에게 있습니다. 단, 참고한 논문의 저작권은 원저자에게 있으므로 인용 시 반드시 출처를 표기하세요.

**Q4. 데이터 삭제(회원탈퇴)를 원합니다.**
A. **[계정 설정]** → **[회원탈퇴]** 또는 privacy@academi.ai 로 요청해주세요. 요청 접수일로부터 7일 내 완전 삭제됩니다.

**Q5. 오프라인에서도 사용할 수 있나요?**
A. 편집 중 일시적 네트워크 단절은 브라우저 로컬 스토리지에 보관되었다가 재연결 시 동기화됩니다. 완전한 오프라인 모드는 현재 지원하지 않습니다.
